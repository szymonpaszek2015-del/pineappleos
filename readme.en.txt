RevoltOS is a stable and secure Ubuntu-based operating system designed to provide a comfortable and intuitive user experience. Our goal is to combine simplicity with performance so you can focus on what truly matters. The system is scheduled for release on October 13, 2025.

System Requirements
To ensure smooth and efficient operation, your computer must meet the following requirements

Processor Multi-core 64-bit processor clocked at 2 GHz or faster

RAM Minimum 2 GB (4 GB or more recommended)

Disk Space At least 20 GB of free disk space

Graphics Card Graphics card supporting DirectX 9 or OpenGL 3.3

Display 800x600 resolution or higher

Installation
Installing the system is simple and takes only a few minutes.

Download the ISO image Visit our website and download the latest version of the system.

Prepare the installation media

Use a program like Rufus (for Windows) or BalenaEtcher (for Linux and Windows, but Reccomennded for Linux) to create a bootable USB drive from the downloaded ISO file.

Run the installer

Insert the USB drive into your computer and restart it. Depending on your computer, you may need to enter the BIOSUEFI to change the boot order.

When the installer screen appears, follow the instructions to select your language, time zone, and keyboard layout.

Install

Choose where you want to install the system (e.g., entire disk or a selected partition) and follow the installer prompts until the process is complete.
